# My Portfolio (Angular)

This project is now an Angular application with Tailwind CSS styling.

## Run Locally

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm start
```

The app runs with Angular CLI (`ng serve`).

## Build

Create a production build:

```bash
npm run build
```

Build output is generated in `dist/my-portfolio`.

## Deploy

```bash
npm run deploy
```
